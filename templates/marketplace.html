Manual Installation

Modules can also be installed manually by cloning the repo into modules/ and adding a replace directive to core/go.mod: 

# Clone the module
git clone https://github.com/Muxcore-Media/jellyfin modules/jellyfin

# Add to go.mod
cd core
go mod edit -require github.com/Muxcore-Media/jellyfin@v0.0.0
go mod edit -replace github.com/Muxcore-Media/jellyfin=../modules/jellyfin

# Rebuild
go build -tags default ./cmd/muxcored
