Configuration Guide

Settings are configured via environment variables with the MUXCORE_ prefix. Set these in your docker-compose.yml or .env file, then restart MuxCore for changes to take effect.

Example .env:

# Core
MUXCORE_STORAGE_PATH=/data/media

# qBittorrent
MUXCORE_QBITTORRENT_ADDR=http://localhost:8080
MUXCORE_QBITTORRENT_USER=admin
MUXCORE_QBITTORRENT_PASS=yourpassword

# Jackett
MUXCORE_JACKETT_ADDR=http://localhost:9117
MUXCORE_JACKETT_APIKEY=yourapikey

# Jellyfin
MUXCORE_JELLYFIN_URL=http://localhost:8096
MUXCORE_JELLYFIN_APIKEY=yourapikey
{{end}}
